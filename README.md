# Greenspark

## How to start the app

```sh
cd greenspark-vue
pnpm install
```

### Start the app in dev 

```sh
pnpm dev
```

### Build the app

```sh
pnpm build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
pnpm test:unit
```

## Tech stacks
* Vue3
* Vuex for state management
* TypeScript
* Vitest for testing
* Tailwindcss for styling
